add
~~~


.. function:: add(evm: ethereum.banana.vm.Evm) -> None
   :noindexentry:

   Adds the top two elements of the stack together, and pushes the result back
   on the stack.

   :param evm: The current EVM frame.

   :raises ethereum.banana.vm.error.StackUnderflowError: If `len(stack)` is less than `2`.
   :raises ethereum.banana.vm.error.OutOfGasError: If `evm.gas_left` is less than `3`.
